# Library Management System (Java)

A simple console-based Library Management System built using Java, File Handling, and Serialization.

# Features

* Add new books
* Issue books to students
* Return books
* View all books
* Stores data permanently using `books.dat`

# How to Run

1. Clone the repository
2. Open project in any IDE (Eclipse/VS Code)
3. Run `Library_Management.java`

# Technologies Used

* Java
* Serialization
* File Handling
