***Restaurant Billing System (Java)**

A simple console-based Restaurant Billing System built using Java and Collections Framework.

**Features**

Add new menu items

Remove menu items

Take customer orders

Calculate bills with GST

Generate itemized receipts

Dynamic menu management

**How to Run**

Clone the repository

Open the project in any IDE (Eclipse/IntelliJ/VS Code)

Run Main.java

**Technologies Used**

Java

Collections (HashMap, ArrayList)

Console I/O
